# CodeAlpha Login & Registration System

## Overview

The CodeAlpha Login & Registration System is a C++ console-based authentication application developed as part of the CodeAlpha C++ Programming Internship.

The project demonstrates user authentication, file handling, input validation, and secure credential management concepts.

---

## Features

* User Registration
* User Login Authentication
* Duplicate Username Detection
* Password Validation
* File-based Credential Storage
* Error Handling for Invalid Inputs

---

## Technologies Used

* C++
* File Handling
* Functions
* String Manipulation
* Input Validation

---

## How It Works

1. Users can register a new account.
2. The system validates username and password requirements.
3. User credentials are stored securely using file handling.
4. Registered users can login using their credentials.
